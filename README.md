# 🌍 Peace Wordle

A simple educational Wordle-style game built in Unity that encourages players to explore words related to peace, learning, sustainability, and global citizenship.

## 🎮 Features

- Classic 5-letter Wordle gameplay
- Educational-themed word list
- Clean and minimal user interface
- Randomized word selection
- Keyboard and mouse support

## 🛠️ Built With

- Unity
- C#
- TextMesh Pro

## 📚 About

This project explores how familiar word puzzle mechanics can be combined with educational themes to create an engaging learning experience.

The game focuses on positive vocabulary related to education, cooperation, the environment, and global awareness while keeping gameplay simple and accessible.

## 🚀 Future Improvements
